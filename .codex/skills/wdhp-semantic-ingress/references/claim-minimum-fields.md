# Claim Minimum Fields

Minimum promoted claim fields:

- `claim_id`
- `wave_id`
- `claim_scope`
- `claim_target_id`
- `sources_read`
- `open_questions`
- `compiled_into`
- `submitted_at`
- `semantic_findings`

Minimum semantic finding fields inside the promoted claim:

- `semantic_id`
- `semantic_node_type`
- `title`
- `summary`
- `business_conclusion`
- `primary_source_refs`
- `supporting_source_refs`
- `semantic_authority`
- `durable_target_pages`
- `confidence`
- `last_verified`
- `open_questions`
- `non_equivalent_to`
