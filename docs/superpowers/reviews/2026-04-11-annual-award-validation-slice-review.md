### Important (Should Fix)

### Minor (Nice to Have)

1. **`evidence_refs` naming is fragile**
   - File: `src/work_data_hub_pro/capabilities/identity_resolution/service.py:102`
   - Issue: Format `f"identity:{method}:{company_name}"` could silently break if callers parse expecting a different format
   - Impact: Low - internal usage only
   - Recommendation: Consider a dataclass with explicit fields

2. **CompatibilityCase lacks anchor_row_nos linkage**
   - File: `src/work_data_hub_pro/governance/compatibility/models.py`
   - Issue: `CompatibilityCase` has no field to track which anchor_row_no values contributed to the difference
   - Impact: Low - trace events can be regenerated by re-running the slice
   - Recommendation: Add optional `involved_anchor_row_nos: list[int]` field

3. **No write validation on evidence_index**
   - File: `src/work_data_hub_pro/apps/orchestration/replay/annual_award_slice.py:143`
   - Issue: If evidence directory isn't writable, slice continues silently without indexed evidence
   - Impact: Low - failure would surface on next evidence retrieval
   - Recommendation: Add check after `evidence_index.index_trace_events()` to verify file created

4. **Test fixture drift detection gap**
   - Files: `tests/replay/test_annual_award_slice.py`, `tests/replay/test_annual_award_explainability_slo.py`
   - Issue: Identical content written in tests matches reference files - if reference files change, tests don't catch drift
   - Impact: Low - governance doc test `test_annual_award_governance_docs.py` provides coverage
   - Recommendation: Consider periodic manual review of fixture alignment

---

## Recommendations

1. **Add trace event linkage to CompatibilityCase** - Track which rows contributed to a compatibility diff for faster debugging.

2. **Validate evidence_index writes** - Add explicit verification that indexed evidence files are created successfully.

3. **Consider adding a drift detection test** - Periodically verify fixture files in `reference/historical_replays/annual_award/` remain aligned with what the replay slice produces.
